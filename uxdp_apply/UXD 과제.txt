 
01. UI Design부문비주얼 유저 인터페이스 디자인 부문 
 
과제 네이버 또는 한게임의 새로운 컨텐츠를 제안하고 그와 함께 UI Design 스타일을 새롭게 구성하고 제안하라. 단, 네이버 또는 한게임의 기존 컨텐츠를 유지하면서 새로운 UI Design 스타일을 제안하는 것도 가능함. 각 개인의 창의성이 드러나도록 할 것. 
제출물 페이지 디자인 (형식은 자유 html,flash,movie도 가능)
컨텐츠 및 디자인 제안서 (ppt형식, 2페이지 이내)
UI design 포트폴리오 (본인이 작업한 작품에 한함)

예상 질문

1. UI 디자인인가, HCI 인가?

인지 공학이나 기획에 대한 전문지식을 갖추지는 못했지만, 디자이너도 사용성 개선에 대한 생각을 
항상 인식하고 디자인을 해야 한다는 것. 단순 미적감각만 가지고는 UI 디자인은 할수 없다.

물론 책의 겉표지도 중요하지만 책의 내용이 더욱 중요하다고 생각

스타일 이전에 사용성이라고 생각

새로운 스타일의 디자인도 좋은 디자인이지만 사용중인 컨텐츠에 어울리는 조그만 아이콘도 같은 디자인이라고 생각

감각적이면서 이용하기 쉬운 인터페이스 디자인을 목표로 공부


2. 좋은 인터페이스란?

사용자의 소중한 시간을 절약해 주는 인터페이스
참신한 인터페이스는 기존의 인터페이스에 비해 인지하는 시간이 걸리긴 하지만
사용자가 그 시간이 아깝다고 생각을 못하도록, 좋은 경험을 했다라는 느낌 들도록
다시 사용했을때는 편리성까지 느낄수 있는 인터페이스
그저 어렵기만한 인터페이스는 시간을 뺏긴다고 생각하고 사용자의 집중을 떨어 트림. 컨텐츠를 죽이는일

사이트의 성격에 따라
정보전달 사이트
우선은 이용이 편한것이 첫째, 정보의 길잡이, 빠른 접근, 택스트 및 단순한 이미지

즐기는 사이트
개성이 살아있으면서 이용이 용이
반복 사용시 지루해짐, 많은 변화 주어야 함
